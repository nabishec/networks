# Глава 3. Канальный уровень — карта

Tanenbaum, гл. 3 (стр. PDF 242–311). Передача данных **между двумя соседями по каналу**: формирование фреймов, обнаружение и коррекция ошибок, надёжная доставка через скользящее окно.

## Базовые понятия

- [[Канальный уровень]] — задачи: framing, error/flow control.
- [[Фрейм]] — единица L2.
- [[Формирование фреймов]] — методы: подсчёт байтов, флаг+стаффинг, физ. сигнал.
- [[Битстаффинг]] — HDLC-приём против ложных флагов.

## Обнаружение и коррекция ошибок

- [[Корректирующие коды — Хэмминг]] — расстояние Хэмминга, исправление 1 ошибки.
- [[Код Рида-Соломона]] — пакетные ошибки, основа CD/QR.
- [[LDPC и сверточные коды]] — современные FEC.
- [[CRC — циклический избыточный код]] — детектирование через полином.
- [[Internet checksum]] — 16-битная сумма IP/TCP/UDP.

## Элементарные протоколы

- [[Stop-and-Wait]] — простейший ARQ.
- [[Скользящее окно]] — много фреймов в полёте.
- [[Go-Back-N]] — окно отправителя > 1, получателя = 1.
- [[Selective Repeat]] — оба окна > 1.
- [[Piggybacking]] — встраивание ACK в обратный поток.
- [[ARQ]] — Automatic Repeat reQuest.

- [[DOCSIS]] — канальный аспект (см. гл. 4).
