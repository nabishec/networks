# Networks Vault — главная точка входа

Связанная база знаний по компьютерным сетям, построенная на основе:
> Tanenbaum, Feamster, Wetherall. **Компьютерные сети**. 6-е изд. (Питер, 2023, ISBN 978-5-4461-1766-6).

## Как пользоваться

Это **связанная** база — каждое понятие ссылается на 3+ соседних. Не нужно читать линейно. Три способа навигации:

1. **По главам книги** — последовательное прохождение → [[#Карты по главам]].
2. **По уровням стека** — сравнение и cross-cutting → [[by-layer|Карта по уровням]].
3. **По маршрутам обучения** — целевое освоение → [[learning-paths|Маршруты]].

Заметки атомарные: одна идея = один файл. Каждая содержит TL;DR, проблему, механизм, пример, связи и подводные камни. Цель — за <30 секунд понять понятие.
