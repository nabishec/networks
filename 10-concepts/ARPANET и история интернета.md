# ARPANET и история интернета

## TL;DR
Хронология: **1962** Бэран — идея коммутации пакетов; **1969** ARPANET — 4 узла (UCLA, Stanford, UCSB, Utah); **1972** email; **1973** TCP (Cerf, Kahn); **1983** TCP/IP стал стандартом ARPANET; **1986** NSFNET-backbone; **1989** Tim Berners-Lee — WWW; **1993** Mosaic browser; **1995** коммерциализация (приватизация NSFNET). 4 архитектурных решения, ставших ДНК интернета: коммутация пакетов, отказоустойчивость через redundancy, end-to-end-принцип, открытые стандарты.

## Какую проблему решает
История объясняет «почему интернет именно такой». Многие странности (TCP вместо чего-то лучшего, IPv4-32-бит, BGP-policy) — наследство решений 1970-80-х. Понимание истории = понимание ограничений и trade-offs, на которых стоит современная сеть.

## Как работает (хронология)

| Год | Событие |
|---|---|
| 1962 | Paul Baran (RAND): packet switching для отказоустойчивых военных коммуникаций |
| 1965 | Donald Davies (UK NPL): «packet» как термин |
| 1969 | ARPANET: 4 IMP (Interface Message Processors), первое сообщение «LO» |
| 1972 | Ray Tomlinson — email, символ `@` |
| 1973–74 | Cerf & Kahn — TCP/IP design |
| 1981 | RFC 791 (IP), 793 (TCP) |
| 1983 | ARPANET cutover на TCP/IP («Flag Day»). DNS rfc 882 |
| 1986 | NSFNET — первая backbone из пяти суперкомпьютерных центров |
| 1988 | Morris Worm — первый интернет-инцидент |
| 1989 | Tim Berners-Lee (CERN) предлагает WWW |
| 1991 | NSFNET снимает запрет на коммерческий трафик |
| 1993 | Mosaic browser (NCSA) — массовый WWW |
| 1995 | NSFNET decommissioned; коммерческие ISP занимают backbone |
| 1998 | ICANN; Google основан |
| 2003 | IPv6 deployment начало |
| 2010s | Mobile-internet доминирует; HTTPS повсеместно |
| 2020s | QUIC/HTTP/3, post-quantum prep |

## Ключевые архитектурные наследия

1. **Коммутация пакетов** (Бэран): отказоустойчивость через distributed routing — хрущёв «не положишь ядерную бомбу на правильный коммутатор».
2. **End-to-end argument** (Saltzer, Reed, Clark, 1984): надёжность на endpoints, сеть тупая.
3. **Открытые RFC**: «rough consensus and running code» — код важнее красивых документов.
4. **IPv4 32 бита** (1981): «больше чем достаточно на foreseeable future» — на самом деле кончились в 2010s.

## Пример
**Почему BGP по умолчанию доверчив:** ARPANET был между академическими/военными — все доверяли друг другу. С коммерциализацией доверчивость осталась, но угрозы выросли. Отсюда RPKI и BGP-hardening сегодня.

**Почему IP без шифрования:** 1980s — шифрование было экспортно-контролируемым (US ITAR). TLS появился позже как отдельный слой.

## Подводные камни
- **Миф о военном происхождении:** ARPA = Advanced Research Projects Agency (DoD), но ARPANET строился академическими лабораториями для **research-collaboration**. «Военный сетевой проект для выживания ядерной войны» — миф.
- **Internet ≠ WWW.** WWW (1989+) — одно из приложений интернета (1969+).
- **Linux/Open Source** не был частью ARPANET — параллельная история, сошлась в 1990s.
