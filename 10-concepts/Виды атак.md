# Виды атак (taxonomy)

## TL;DR
Главные типы атак на сети: **reconnaissance** (разведка: scan, fingerprint), **eavesdropping** (пассивное подслушивание), **spoofing** (подделка источника: ARP/DNS/IP/email), **MITM** (man-in-the-middle), **DoS / DDoS** (отказ в обслуживании), **malware** (ransomware, worms, trojans), **phishing** (social engineering), **data theft**. Каждая атакует одну или больше из [[Триада CIA|CIA]]-целей.

## Какую проблему решает
Понимание категорий атак — основа для проектирования защиты. Без этого — играешь в whack-a-mole, не видя системно.

## Как работает

### Reconnaissance — разведка
- **Port scanning** (nmap) — какие сервисы открыты.
- **OS fingerprinting** — какой ОС.
- **Banner grabbing** — версии сервисов.
- **OSINT** — публичная инфо в whois, github, social media.
Цель — найти **vulnerabilities** для атаки.

### Eavesdropping (пассивное)
- **Sniffing** в LAN (Wireshark) — слушать broadcast или own port.
- **Wi-Fi-sniffing** — публичный Wi-Fi без шифрования.
- **Tap на провайдере** — лом-вмешательство в физический канал.
**Защита:** TLS, IPsec, шифрованный Wi-Fi.

### Spoofing — подделка
- **IP spoofing** — фальшивый src IP. Защита: uRPF, BCP38 фильтрация.
- **ARP spoofing** ([[ARP spoofing]]) — MITM в LAN.
- **DNS spoofing** ([[DNS spoofing]]) — подменить ответ. Защита: DNSSEC.
- **Email spoofing** — фальшивый From. Защита: SPF/DKIM/DMARC.

### MITM
Атакующий встаёт между клиентом и сервером, может **читать** и **изменять**.
Защита: end-to-end шифрование с **certificate pinning** или authentic key exchange.

### DoS / DDoS
- **DoS:** перегрузить сервис ресурсов (одиночный источник).
- **DDoS:** распределённая, тысячи источников (botnet).
- **Amplification:** маленький запрос → большой ответ (DNS, NTP, memcached) → жертве.
- **Application-layer:** Slowloris, HTTP-flood — труднее фильтровать.
**Защита:** rate-limit, CDN/anti-DDoS, scrubbing centers.

### Malware
- **Worm:** само-распространяется через уязвимости (1988 — Morris worm).
- **Virus:** распространяется через user action (открытие файла).
- **Trojan:** маскируется под legitimate.
- **Ransomware:** шифрует файлы, требует выкуп.
- **Botnet client:** превращает машину в часть botnet.

### Phishing
- Social engineering: фишинговое письмо «от банка» с ссылкой на фальшивый сайт.
- **Spear-phishing:** target'ed на конкретное лицо/компанию.
**Защита:** обучение пользователей, U2F-ключи, MFA.

### Data theft
- **Insider:** сотрудник копирует БД на USB.
- **External:** взлом → exfil через DNS-tunneling, encrypted channels.
**Защита:** DLP, audit, encryption at rest, MFA.

## Пример
**SolarWinds attack (2020):** supply-chain — заражение build-system → trojan в обновлении SolarWinds Orion → 18 000 организаций → exfil. Категория: data theft + APT (Advanced Persistent Threat).

**WannaCry (2017):** ransomware-worm — exploit EternalBlue в Windows SMB → distribution + encryption. Mix of malware + DoS (через encryption).

## Подводные камни
- **Modern attacks комбинированы:** APT использует phishing для initial access → reconnaissance → privilege escalation → exfiltration. Не один метод.
- **Insider threat** часто недооценен — большая часть утечек проходит через разрешённые credentials.
- **Zero-day** уязвимости — без CVE, без патча. Защита — defense-in-depth.
