# Цифровой звук — кодеки

## TL;DR
Звук → биты: **дискретизация** + **квантование** + **сжатие**. Без сжатия (PCM, CD-quality 1.4 Мбит/с) много байт. Сжатие психоакустическое (отсекает то, что ухо не различает): **MP3** (1995), **AAC** (1997), **Opus** (2012, default WebRTC). Целевые битрейты: голос 8–32 кбит/с, музыка 96–256 кбит/с, hi-fi 320 кбит/с+.

## Какую проблему решает
PCM CD-качества: 44.1 кГц × 16 бит × 2 канала ≈ 1.4 Мбит/с. Час музыки = 600+ МБ. Слишком много для интернета, особенно мобильного. Кодеки сжимают **в 10-50 раз** с минимальной потерей качества за счёт **психоакустики**: ухо нечувствительно к определённым частотам в присутствии других, к слабым звукам близко по времени к громким.

## Как работает

**PCM** — несжатый baseline ([[PCM — импульсно-кодовая модуляция]]).

**Психоакустическое сжатие (lossy):**
1. **FFT** — звук → частотные коэффициенты.
2. **Masking model** — какие коэффициенты «не слышны».
3. **Quantization** — грубо округлить инвидимые.
4. **Huffman / arithmetic coding** — сжать.

**Главные кодеки:**

| Кодек | Год | Применение | Bitrate | Свойства |
|---|---|---|---|---|
| **PCM (G.711)** | 1972 | телефония | 64 кбит/с | без сжатия, low CPU |
| **MP3** | 1995 | музыка/iPod | 96–320 кбит/с | классика, патенты истекли 2017 |
| **AAC** | 1997 | YouTube, Apple, mobile | 64–256 кбит/с | лучше MP3, сложнее |
| **Opus** | 2012 | WebRTC, видеоконф. | 6–510 кбит/с | универсальный, открытый, низкая задержка |
| **G.729** | 1996 | VoIP | 8 кбит/с | специально для голоса |
| **Speex** | 2003 | open VoIP | 2–44 кбит/с | заменён Opus |

**Bitrate vs качество:**
- 32 кбит/с Opus — близко к телефону.
- 96 кбит/с Opus — индастриальный стандарт для подкастов.
- 192 кбит/с MP3 — «high quality».
- 320 кбит/с MP3 — почти неотличимо от CD.

## Пример
- **Spotify:** 96/160/320 кбит/с в Vorbis или AAC.
- **YouTube Music:** AAC 128–256 кбит/с.
- **WebRTC video calls:** Opus 24–48 кбит/с — задержка <20 мс.
- **Discord voice:** Opus 64 кбит/с.

## Связи
- **Базируется на:** [[PCM — импульсно-кодовая модуляция]] (raw), психоакустика.
- **Используется в:** [[VoIP и SIP]] (real-time), [[HLS и DASH]] (стриминг), [[RTP]] (transport).
- **Соседи по уровню:** [[Цифровое видео — кодеки]] — параллельная иерархия.
- **Противопоставляется:** lossless форматы (FLAC, ALAC) — больше байт, но без потерь.

## Подводные камни
- **Lossy** — каждое перекодирование теряет качество. Архивы → lossless (FLAC).
- **Latency vs efficiency** trade-off: long lookahead = лучше сжатие, но больше задержки. Opus имеет low-delay режимы.
- **Patent landscape** менялся: MP3-патенты истекли 2017 → можно freely. AAC всё ещё имеет патентные требования. Opus — свободный.
